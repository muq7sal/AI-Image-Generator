## Project overview
A local text-to-image generator using Stable Diffusion (open-source), with a Streamlit web UI, adjustable generation parameters, prompt engineering support, content filtering, watermarking, and saving/export of images with metadata.

- Text prompt ‚Üí images (multiple)
- Negative prompt support

- Watermarking and simple prompt filtering
- Images saved with metadata JSON
